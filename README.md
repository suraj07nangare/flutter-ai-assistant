# 🤖 Flutter AI Assistant

A modern **cross-platform AI Assistant** built with **Flutter** that enables users to interact with AI through natural language conversations and AI-powered image generation. The application provides a clean, responsive, and intuitive user experience while integrating powerful AI capabilities using OpenAI APIs.

---

## ✨ Features

* 💬 AI-powered conversational chatbot
* 🖼️ AI image generation from text prompts
* ⚡ Fast and responsive cross-platform application
* 📱 Beautiful Material Design user interface
* 🔄 Real-time API communication
* 🌙 Modern and intuitive user experience
* 📲 Android and iOS support

---

## 🚀 Getting Started

### Prerequisites

* Flutter SDK
* Android Studio or VS Code
* Dart SDK
* OpenAI API Key

### Installation

1. Clone the repository

```bash
git clone https://github.com/suraj07nangare/flutter-ai-assistant.git
```

2. Navigate to the project

```bash
cd flutter-ai-assistant
```

3. Install dependencies

```bash
flutter pub get
```

4. Configure your OpenAI API key.

5. Run the application

```bash
flutter run
```
